# Password Strength Checker

An intermediate-level C++ console application for analyzing password strength, built as a second-year B.Tech IT project.

## Features

### Core Requirements
- Password strength checking based on 5 criteria
- Strength levels: Weak, Medium, Strong, Very Strong
- Detailed analysis report
- Strength score out of 100
- Improvement suggestions
- Modular function-based design
- Input validation and error handling
- File logging of analysis results

### Additional Features
- Session-based password history (prevents reuse warning)
- Strong password generator
- Menu-driven interface
- Results stored in `password_analysis.log`
